# DST Manager

## 文件结构

- _docs
- cmd
- localization
- tui
- testdata
    - 测试用文件
    - 里面可以放Steam文件, 饥荒服务端, 游戏存档文件夹
    - 注: 代码的测试放在各个代码同一个文件夹里, 文件名为`xxx_test.go`
- templates

## commit前缀

- feat: 添加新功能
- fix: 修复bug
- style: 增减空白等等, 对功能没有影响的变更
- docs: 添加/变更文档
- perf: 性能改善
- refactor: 重构(不添加新功能, 也不修bug)
